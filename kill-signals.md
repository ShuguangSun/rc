# Signals

## List processes

    ps auxf

## kill

    kill -HUP $( cat /var/run/your_process.pid )
    kill -SIGUSR1 $( cat /var/run/your_process.pid )

## pkill

Kill process by matched command regex

    pkill --full command-match-regex
